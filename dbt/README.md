Welcome to my dbt project. This project is for trying out dbt and getting hand-on experience with it.

## Setup environment

Use the following code

```
python3 -m venv dbt_pyenv
```

## Using the project

Try running the following commands:
- dbt debug
- dbt run
- dbt test
- dbt docs generate
- dbt docs serve
